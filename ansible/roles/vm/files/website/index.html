Descripción del Proyecto

Esta página web ha sido desplegada como parte del Caso Práctico 2 para el Curso Avanzado de DevOps en la UNIR (Universidad Internacional de La Rioja).

El objetivo de este ejercicio es demostrar la automatización de la infraestructura y el despliegue de aplicaciones usando prácticas y herramientas modernas de DevOps.

Tecnologías Utilizadas

Terraform

Se ha utilizado para aprovisionar la infraestructura en Azure, incluyendo máquinas virtuales y recursos de red.

Ansible

Se ha utilizado para la gestión de configuración y el despliegue automatizado de la aplicación contenerizada.

Podman

Se ha utilizado como runtime de contenedores para ejecutar Nginx en un entorno seguro y sin demonio.

Nginx

Servidor web ligero usado para servir esta página estática dentro de un contenedor.
